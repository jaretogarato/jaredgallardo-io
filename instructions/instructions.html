1 . 2 - Change the Colors

1.2.1 - HTML Theme Text Color
 
The theme text color can be changed in the index.html file. Any HTML tag that has a class called "text-color" (as defined in section 4.0 of the style.css file) will become that color... (text-color is the theme color for the entire website, only affecting text with that class)

index.html

<span class="text-color">I am red because the style.css file contains my class as seen below...</span>
1.2.2 - CSS Theme Text Colors
 
By default, the main theme color is set to #ffee00 (yellow) with the class .text-color in style.css...

See section 4.0 in style.css

.text-color {
    color:#ffee00;
}
NOTE: There are other color classes such as "white", "black", "red", "green", "yellow", etc... If a color you want doesn't have a class specified for it, just create it in the style.css file, and then you can use it in your index.html file...

style.css

.orange {
    color:#ff9900;
}

index.html

<div class="orange">I am orange because the style.css file defines my class.</div>
 
<h1 class="orange">I am also orange because I am using the same class.</h1>
NOTE: The "color picker" you saw in the preview version is available in the "preview" folder included in your downloaded theme if you would like to use it. See 1.2.4 below for more information on changing the colors that appear in the "color picker" within the header of the preview version...
1.2.3 - Background Colors
 
The theme background color is set in a class called ".bg-color" in the style.css file. Changing this color will affect every background-color in your index.html file (with a few exceptions, such as #main border-bottom in style.css, which would have to be updated manually).

See section 4.0 in style.css

.bg-color {
    background-color:#ffee00;
}

index.html

<div class="bg-color">My background is blue because the style.css file defines my class.</div>

    ...just make sure you use the same class in your index.html file, such as...

You can also re-use the styles already in place, either by replacing the background image in the "images" folder (like replacing bg_home.jpg), or by changing the file name in the style.css file for the home section like background-image:url(images/my_awesome_photo.jpg); - These same rules also apply to about and contact.
 
background-attachment:fixed; keeps the photo in a "fixed" position so it will not scroll with the rest of the page (some smartphones will scroll the image anyway, and then "snap" it back into a new position... welcome to my world). The default setting of "scroll" (instead of "fixed") will allow the background image to scroll freely with the rest of the page... background-attachment:scroll; - Please note that some old browsers may not support fixed backgrounds.
 
background-position:center center; controls the position of the photo. You could try background-position:left top; or background-position:center bottom; or any combination (first parameters are left, center, or right, while the second parameter id top, center, or bottom). You can also set more precise positioning like background-position:-100px -300px;. If I don't stop here, I will end up teaching you PhotoShop.

1 . 6 - Change the Crew Section Photos and Social Networking Icons

1.6.1 - Change the Crew Section Photos
 
The quickest, easiest way to change photos in the crew section is to just replace the existing photos in the "images" folder, crew1.jpg and crew2.jpg.
 
For example, replace crew1.jpg with your own photo and use the same name, crew1.jpg ...thereby deleting the old one.
1.6.2 - Crew Photo Sizes
 
Images for crew photos should be around 500 to 900 pixels wide (a similar height is helpful, but not required), with a resolution of 72 pixels per inch. They need to be a little large, because they will be full width on some smartphones and small tablets. They need to look "sharp", because smaller photos will look blurry when they are stretched.
 
If you don't have any photo editing software, contact us through http://www.themeforest.net/user/egobuilders and we can help you. A small fee may apply for resizing or editing images and photos.
1.6.3 - Change the CSS Classes for Crew Photos
 
If you need to change the CSS coding for the crew photos, here's where to do it...
NOTE: The properties in .crew-photo (in the style.css example below) apply to all crew photos, so changing the properties in this class will affect them all of the crew photos the same way. Feel free to copy this class and create a new one for each photo (like .crew-photo-1 and .crew-photo-2, etc.), and don't forget to update the HTML, in the example below this style.css example. In responsive.css, the height and margin for .crew-photo are often adjusted for different screen sizes (ergo, responsive).

See the very bottom of section 6.4 in style.css

7 . 2 - Responsive

The file that manages all of the "responsive" aspects of the template is responsive.css - You may modify classes, id's, or selectors in this file if you are familiar with responsive design, otherwise, modifying that file is discouraged.
 
Basically, the widest screen settings are defined at the top, and the least wide screen settings are defined as you go down the stylesheet. A setting for a wider screen will only affect every width below it, unless another setting is defined to overwrite it within a lower width. For example, for all screen widths under 1400 pixels, a class may have a font-size of 20 pixels (20px) ...and that will be the font-size for everthing below it in that same class, unless you set that class to be, let's say, 15 pixels (15px) for a smaller screen width. I prefer pixels (px) instead of points (pt) because points really apply to printing, while pixels are obviously related to screen resolution.
